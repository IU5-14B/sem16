# Семинар 16

Проект содержит простые реализации задач со страницы "Семинар 16":

1) **Медиана массива** через предварительную сортировку  
   - Алгоритм: отсортировать массив, взять элемент с индексом `n/2` (целая часть).  
   - Сложность: `O(n log n)` по времени, `O(1)` доп. памяти (если сортировка in-place).

3) **Пересечение двух множеств** `A` и `B`  
   - (a) Грубая сила: двойной цикл `O(n*m)`  
   - (b) Через сортировку + два указателя: `O(n log n + m log m + n + m)`

4) **Минимум и максимум в массиве**  
   - (a) Через сортировку: `O(n log n)`  
   - (b) Сравнение:
     1. Грубая сила (1 проход): `O(n)`
     2. Через сортировку: `O(n log n)`
     3. Декомпозиция (попарно): `O(n)` и меньше сравнений (примерно `~ 3n/2`)

## Сборка и запуск

```bash
rm -rf build && cmake -S . -B build && cmake --build build && ./build/seminar16
```

## Формат ввода

Программа интерактивная: предложит меню и попросит ввести массивы.
Можно выбрать **ручной ввод** или **случайную генерацию**.

> Код написан максимально прозрачно и без «сложных конструкций».
